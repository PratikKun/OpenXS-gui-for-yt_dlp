One-Command Installation

Installs to system, creates desktop entry, and adds to application menu:

# Install OpenXS Video Downloader
curl -sSL https://raw.githubusercontent.com/PratikKun/OpenXS-gui-for-yt_dlp/main/install.sh | bash

Or download and run manually:

wget https://raw.githubusercontent.com/PratikKun/OpenXS-gui-for-yt_dlp/main/install.sh
chmod +x install.sh
./install.sh

What it does:

  • ✓ Detects your Linux distribution automatically
  • ✓ Installs all required dependencies (Qt5, yt-dlp, etc.)
  • ✓ Builds and installs the application to /usr/local/bin
  • ✓ Creates desktop entry for all desktop environments
  • ✓ Adds to application menu under "Utilities"
  • ✓ Works with GNOME, KDE, XFCE, MATE, Cinnamon, and more
